HardbassSMP 8.0

yes, we're really on the eighth iteration, i counted


Welcome to New New New New New New New New New Whitlawburn!

The server returns! This time on a dedicated machine, and hopefully more stable for it. This site contains general info on the server, a link to the PrismLauncher instance, a mod list if you wanna set your client up manually for whatever reason, and a cool background.

As always the server is quite heavily modded, and you need a good number of these mods installed on your end or the server won't let you in. Below you'll find a button to download a PrismLauncher instance with all the mods and such already in there, and below even that you'll find a list of every mod and the exact versions being used in case you want to set things up manually.

Note for strangers:
This is a private server, and it has a whitelist. If you're not one of my pals, you're not getting added to the whitelist, so if you stumbled upon this site looking for a new server to play on, keep looking.
